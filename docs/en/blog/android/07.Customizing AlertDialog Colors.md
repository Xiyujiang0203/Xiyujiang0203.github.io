# Customizing AlertDialog Colors

Title and Message can be styled with a custom TextView.

```
private void showAlertDialog() {

        AlertDialog.Builder builder = new AlertDialog.Builder(MyDialogActivity.this,R.style.MyAlertButton);//pass style to change button color
        builder.setTitle("");
        TextView textView = new TextView(this);
        textView.setText("Are you sure you want to sign out?");
        textView.setTextSize(14);
        textView.setTextColor(Color.parseColor("#999999"));
        textView.setGravity(Gravity.CENTER_VERTICAL);
        textView.setPadding(50,50,10,10);
        
        //custom TextView color
        builder.setCustomTitle(textView);
        builder.setCancelable(false);
        builder.setPositiveButton("Sign Out", new DialogInterface.OnClickListener() {
            @Override
            public void onClick(DialogInterface dialogInterface, int i) {
                ActivityCollector.finishAll();
                Intent intent = new Intent(MyDialogActivity.this,SignIn.class);
                startActivity(intent);
            }
        });
        builder.setNegativeButton("Cancel", new DialogInterface.OnClickListener() {
            @Override
            public void onClick(DialogInterface dialogInterface, int i) {
                dialogInterface.dismiss();
            }
        });
        builder.show();

    }
```

Buttons can be styled by passing a theme when creating the AlertDialog.Builder.

```
<resources>

    <style name="MyAlertButton"
        parent="@style/Theme.AppCompat.Light.Dialog.Alert">
        <!--Change AlertDialog button color-->
        <item name="colorAccent">#FF0000</item>
    </style>

</resources>
```
