# 《第一行代码》第三版 · 第十四章 · 高级开发技巧

## 第 14 章 进阶，高级技巧

### 14.1 获取全局的Context

- 普通写法：将获取context的时候交给调用方来实现

```
fun String.showToast(context: Context, duration: Int = Toast.LENGTH_SHORT) {
    Toast.makeText(context,this,duration)
}

fun Int.showToast(context: Context, duration: Int = Toast.LENGTH_SHORT) {
    Toast.makeText(context,this,duration)
}
```

- 优化：定制自己的 Application 类 ：程序启动的时候，系统的这个类会被初始化

1. 定制

```
class MyApplication : Application() {

    companion object{
        lateinit var context: Context
    }

    override fun onCreate() {
        super.onCreate()
        context = applicationContext//因为全局只有一个变量所以不存在内存泄露问题
    }
}
```

1. 清单文件中指定初始化的对象为自己的定制MyApplication

```
<application
    android:name=".MyApplication"
```

1. 优化上面的普通写法

```
fun String.showToast(duration: Int = Toast.LENGTH_SHORT) {
    Toast.makeText(MyApplication.context,this,duration)
}

fun Int.showToast(context: Context, duration: Int = Toast.LENGTH_SHORT) {
    Toast.makeText(MyApplication.context,this,duration)
}
//调用，有问题
"This is Toast".showToast()
```

### 14.2 使用Intent传递对象

#### 14.2.1 Serializable 方式

- 将对象转换成可存储可传输的状态，传输后反序列化为对象

```
//让对象可序列化
class Person :Serializable {
    var name = ""
    var age = 0
}
//传输
val person = Person()
        person.name = "Tom"
        person.age = 20
        val intent = Intent(this, SecondActivity::class.java)
        intent.putExtra("person_data", person)//序列化
        startActivity(intent)
//获取
val person = intent.getSerializableExtra("person_data") as Person
```

#### 14.2.2 Parcelable 方式

- 原理：将对象进行拆解，每一部分都可以传输，传输后重新组合成完整的对象，效率比上一种高

```
class Person : Parcelable {

    var name = ""
    var age = 0
    
    override fun describeContents(): Int {
        return 0
    }
		
		//将对象拆解为小部分
    override fun writeToParcel(parcel: Parcel?, p1: Int) {
        parcel?.writeString(name)
        parcel?.writeInt(age)
    }

		//传输后将每一部分重新组合成对象
    companion object CREATOR : Parcelable.Creator<Person> {
        override fun createFromParcel(parcel: Parcel?): Person {
            val person = Person()
            person.name = parcel?.readString() ?: ""
            person.age = parcel?.readInt()!!
            return person
        }

        override fun newArray(size: Int): Array<Person?> {
            return arrayOfNulls(size)
        }

    }

}

//传递相同
//获取
val person = intent.getParcelableExtra<Person>("person_data") as Person
```

- Kotlin优化:将所有参数写在主构造函数里

```
@Parcelize
class Person(var name:String,var age:Int):Parcelable{

}
```

### 14.3 定制自己的日志工具

```
object LogUtil {
    
    private const val VERBOSE = 1
    private const val DEBUG = 2
    private const val INFO = 3
    private const val WARN = 4
    private const val ERROR = 5
    private val level = VERBOSE
    
    fun v(tag:String,msg:String){
        if(level<= VERBOSE){
            Log.v(tag,msg)
        }
    }
    fun d(tag:String,msg:String){
        if(level<= DEBUG){
            Log.d(tag,msg)
        }
    }
    fun i(tag:String,msg:String){
        if(level<= INFO){
            Log.i(tag,msg)
        }
    }
    fun w(tag:String,msg:String){
        if(level<= WARN){
            Log.w(tag,msg )
        }
    }
    fun e(tag:String,msg:String){
        if(level<= ERROR){
            Log.e(tag,msg )
        }
    }
}
```

- 平时的时候设置level的值就可以打印对应等级以下的日志，当上线的时候就提高level

### 14.5 深色主题

1. 打开手机系统的深色主题

2. 利用Froce Dark，让程序快速适配深色主题

- new directory values-v29 ,添加属性 一般不尽如人意

```
<resources>
    <style name="AppTheme" parent="Theme.MaterialComponents.Light.NoActionBar">
        <!-- Primary brand color. -->
        <item name="colorPrimary">@color/purple_500</item>
        <item name="colorPrimaryVariant">@color/purple_700</item>
        <item name="colorOnPrimary">@color/white</item>
        <item name="android:forceDarkAllowed">true</item>
    </style>
</resources>
```

1. 手动实现

- 设置主题：在浅色情况下默认浅色，深色的时候会调整会深色

```
<style name="AppTheme" parent="Theme.MaterialComponents.DayNight.NoActionBar">
```

- 主题差异型编程，设置toolbar和按钮的深色主题颜色

- new directory values-night color.xml

注：书里的属性名可能和我的不一样，我可以去设置自己的toolbar的硬编码颜色

1. 最好的方式

通过主题来指定背景和文字的颜色

```
android:background="?android:attr/colorBackground"
android:textColor="?android:attr/textColorPrimary"
```

1. 不同主题不同代码，判断是否是深色主题的代码

```
fun isDarkTheme(context: Context): Boolean {
    val flag = context.resources.configuration.uiMode and 
            Configuration.UI_MODE_NIGHT_MASK
    return flag == Configuration.UI_MODE_NIGHT_YES
}
```
