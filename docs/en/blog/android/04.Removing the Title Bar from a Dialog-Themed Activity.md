# Removing the Title Bar from a Dialog-Themed Activity

Use

```
supportRequestWindowFeature(Window.FEATURE_NO_TITLE);
```

before `setContentView`.

Does not work:

```
requestWindowFeature(Window.FEATURE_NO_TITLE);   
```

Custom styles also do not work.
