## 输入代码，将 【官网获取的AK】【官网获取的SK】分别用官网控制台中获取的 API Key 和 Secret Key 替换

```python
# encoding:utf-8
import requests 

# client_id 为官网获取的AK， client_secret 为官网获取的SK
host = 'https://aip.baidubce.com/oauth/2.0/token?grant_type=client_credentials&client_id=【官网获取的AK】&client_secret=【官网获取的SK】'
response = requests.get(host)
if response:
    print(response.json())
```

> 补充：requests可以利用Pycharm 命令 pip install requests 安装，pip是很基础的安装工具，不清楚自行百度

## 运行程序，在Run中就能看到返回的  access_token
