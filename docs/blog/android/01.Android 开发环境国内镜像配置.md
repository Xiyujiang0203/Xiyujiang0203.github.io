# Android 开发环境国内镜像配置

在 `settings.gradle.kts` 或项目级 `build.gradle` 的 `repositories` 中添加：

```gradle
maven { url = uri("https://maven.aliyun.com/repository/public") }
maven { url = uri("https://maven.aliyun.com/repository/google") }
maven { url = uri("https://maven.aliyun.com/repository/central") }
```

Gradle 分发包换源（`gradle/wrapper/gradle-wrapper.properties`）：

```
distributionUrl=https\://mirrors.cloud.tencent.com/gradle/gradle-8.0-bin.zip
```

参考：https://blog.csdn.net/qq_36428821/article/details/113444865

代理配置（如遇 SSL 问题）：

```
systemProp.http.proxyHost=127.0.0.1
systemProp.http.proxyPort=4780
systemProp.https.proxyHost=127.0.0.1
systemProp.https.proxyPort=4780
```
