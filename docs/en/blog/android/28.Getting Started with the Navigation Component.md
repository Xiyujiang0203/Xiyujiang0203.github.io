# Getting Started with the Navigation Component

## Navigation Basics Overview

- NavHost — container for fragments

- Fragment — used to place a small piece of layout on the screen

- NavController — controls fragment switching logic

- NavGraph — graphically configures page navigation logic

## Example

1. Two fragments, each layout has one button and one label

2. Add Navigation dependencies from [developers](https://developer.android.google.cn/jetpack/androidx/releases/navigation?hl=zh_cn)

3. Create resourceFile → navigation (my_nav_graph), connect the two fragments

4. Place NavHostFragment in the main layout

5. Add navigation code in fragments — two ways to achieve the same result

```kotlin
override fun onViewCreated(view: View, savedInstanceState: Bundle?) {
        super.onViewCreated(view, savedInstanceState)
        button1.setOnClickListener {
            val controller = Navigation.findNavController(it)
            controller.navigate(R.id.action_homeFragment_to_detailFragment)
        }
    }
```

```kotlin
override fun onViewCreated(view: View, savedInstanceState: Bundle?) {
        super.onViewCreated(view, savedInstanceState)
        button2.setOnClickListener(Navigation.createNavigateOnClickListener(R.id.action_detailFragment_to_homeFragment))
    }
```

1. Set page transition effects: my_nav_graph.xml → line → Animation → set enterAnim and exitAnim; you can use default animations

2. Set up ActionBar for the second fragment

```kotlin
class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
        //get NavHostFragment first
        val navHostFragment = supportFragmentManager.findFragmentById(R.id.fragmentContainerView) as NavHostFragment
        //then get NavController
        val controller : NavController = navHostFragment.navController
        NavigationUI.setupActionBarWithNavController( this,controller)
    }

    override fun onSupportNavigateUp(): Boolean {
        //get NavHostFragment first
        val navHostFragment = supportFragmentManager.findFragmentById(R.id.fragmentContainerView) as NavHostFragment
        //then get NavController
        val controller : NavController = navHostFragment.navController
        return controller.navigateUp()
    }
}
```

1. Set navigation bar labels: resourceFile → navigation (my_nav_graph) → click fragment → modify label
