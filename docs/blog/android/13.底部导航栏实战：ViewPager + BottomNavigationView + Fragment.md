## 二、实现步骤

### 1.编写布局

```
<!-- 相对布局中 -->
<androidx.viewpager.widget.ViewPager
        android:id="@+id/view_pager"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:layout_above="@+id/bottom_navigation_view"
        />

    <com.google.android.material.bottomnavigation.BottomNavigationView
        android:id="@+id/bottom_navigation_view"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:layout_alignParentBottom="true"
        app:itemIconTint="@color/color"
        app:menu="@menu/item"
        app:itemTextColor="@color/color"
        />
```

### 安卓碎片Fragment学习笔记

### 4.获取ViewPager控件，编写适配器传入视图数据

```
private ViewPager viewPager;
private List<View> views = new ArrayList<>();
viewPager = (ViewPager) findViewById(R.id.view_pager);
//将碎片加载为视图
views.add(View.inflate(MainActivity.this,R.layout.first_message_fragment,null));
        views.add(View.inflate(MainActivity.this,R.layout.second_contact_fragment,null));
        views.add(View.inflate(MainActivity.this,R.layout.third_dong_tai_fragment,null));
  

 viewPager.setAdapter(new PagerAdapter() {
            @Override
            public int getCount() {//翻页视图的页数
                return views.size();
            }

            @Override
            public boolean isViewFromObject(@NonNull View view, @NonNull Object object) {
                return view == object;
            }

            @NonNull
            @Override//在翻到该页的时候执行
            public Object instantiateItem(@NonNull ViewGroup container, int position) {
                container.addView(views.get(position));
                return views.get(position);
            }

            @Override
            public void destroyItem(@NonNull ViewGroup container, int position, @NonNull Object object) {//不在该页的时候执行
                container.removeView((View) object);
            }
        });
```

### 5.翻页视图绑定底部导航栏，翻页的时候底部导航栏会变化

```
 viewPager.addOnPageChangeListener(new ViewPager.OnPageChangeListener() {
            @Override
            public void onPageScrolled(int position, float positionOffset, int positionOffsetPixels) {

            }

            @Override//页面被点击的时候回调
            public void onPageSelected(int position) {
                switch (position){//通过页面的下标来选择对应的底部导航栏
                    case 0:
                        bottomNavigationView.setSelectedItemId(R.id.message);
                        break;
                    case 1:
                        bottomNavigationView.setSelectedItemId(R.id.contact);
                        break;
                    case 2:
                        bottomNavigationView.setSelectedItemId(R.id.dong_tai);
                        break;
                }
            }

            @Override
            public void onPageScrollStateChanged(int state) {

            }
        });
```

### 6.底部导航栏绑定视图

```
 bottomNavigationView.setOnItemSelectedListener(new NavigationBarView.OnItemSelectedListener() {
            @Override
            public boolean onNavigationItemSelected(@NonNull MenuItem item) {
                switch (item.getItemId()){//通过底部导航栏的id来选择对应下标的页面视图
                    case R.id.message:
                        viewPager.setCurrentItem(0);
                        break;
                    case R.id.contact:
                        viewPager.setCurrentItem(1);
                        break;
                    case R.id.dong_tai:
                        viewPager.setCurrentItem(2);
                        break;
                }

                return true;
            }
        });
```

以上便可实现开头图片展示的翻页效果
