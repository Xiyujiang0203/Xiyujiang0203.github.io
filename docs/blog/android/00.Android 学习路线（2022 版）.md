# Android 学习路线（2022 版）

Android 开发旨在基于 Android Studio 平台，通过 Java 或 Kotlin 语言开发出运行在 Android 系统上的应用 APP。

## 一、大一第一学期

### 1. 完整学习 Java SE 内容

包括 Java 基本数据类型、分支、循环结构、数组、类和对象、各种包装类及常用方法、封装、继承、多态、抽象类、接口、异常、IO、线程等。

### 2. 学习《第一行代码》第二版

- 掌握 Android 开发环境的搭建、Android 的项目结构
- 掌握 Activity，学习 Activity 的生命周期和启动模式，学会使用 Intent
- 熟练掌握常用的控件、布局，学会使用 ListView、RecyclerView
- 掌握 Fragment、广播、多媒体
- 了解 SQLite 数据库，学会使用 SharedPreferences、LitePal 数据库

### 3. 阶段考核

完成一款具有明确功能的简易 APP。（作为蓝桥杯考核）

## 二、大一第二学期

### 1. 巩固 Java SE 内容

熟练掌握 Java。

### 2. 完整学习《第一行代码》第二版

- 学会运用 SQLite 数据库、Material Design 设计风格
- 掌握 Android 网络编程技术，学会使用 OkHttp 进行网络请求、利用 GSON 解析 JSON 数据、学会调用接口返回的数据
- 掌握内容提供器、运行时权限
- 掌握 Android 的线程机制、服务

### 3. 阶段考核

完成一款具有实际用途、功能较为完备、项目结构合理、能够体现技术点的 APP。（作为暑期考核）

## 三、大二第一学期

### 1. 学习《第一行代码》第三版

学习通过使用 Kotlin 语言进行 Android 开发。

### 2. 源码与自定义 View

阅读 Framework 层源码，阅读一些高级控件的源码（如 ListView、RecyclerView），尝试自定义 View。

### 3. 进阶知识

学习 Android 的动画等进阶知识，学习部分设计模式。

### 4. 官方文档

阅读 Android 官方文档。

### 5. 阶段目标

具备基本的自主开发能力。

## 四、后续阶段

此后应结合自身实际情况，综合个人技术以及项目经验，自行安排学习计划。

## 推荐书籍

- 《第一行代码》第二版
- 《第一行代码》第三版
- 《Java 从入门到精通》

- 《Kotlin 核心编程》

## 补充笔记

### 大一上 · 环境配置

- 安装 IntelliJ IDEA CE（官网下载）—— Java 编译环境
- 安装 JDK 1.8（可在编译环境内下载）
- 具体环境配置可上 CSDN 找教程，或向学长请教
- Android Studio 官网：https://developer.android.google.cn
- 配置 SDK 环境（一般在安装时可一步到位；若因网络问题无法下载，可向学长索取已下载好的 SDK）

### 大一上 · 学习经验

- 书籍：《Java 从入门到精通》
- 视频：B 站黑马程序员 Java 教学视频（找一个适合自己的）
- 可以跟着敲，但没必要全部都敲，看懂很重要
- 本阶段约 1～2 周完成 Java 基础，进入科协后两周内完成整体任务
- 方向本身不要求做笔记，但科协的笔记任务需完成
- 要求能看懂，快速遗忘没事；用到时再查 CSDN，写着写着就熟悉起来
- 跳着看书有看不懂的风险，注意是否跳过了前面内容
- 用 Typora 记录过程，方便复习和重用代码
- 不要只看书，跑通书中代码很重要
- 不要害怕忘记，学习就是反复循环（例如适配器就可能需要反复看好几遍）

### 寒假

继续学习《第一行代码》。

**建议**：大二开学前跑通书中所有代码，并在 Typora 上保存记录。第十一章、第十五章可以不看；第十四章不用去实现（bug 较多），可学习其中的架构模式。

### 大一下 · 练手项目：WanAndroid 平台

**题目描述**：经过两个学期的安卓学习，开发一个 App 帮助初学者更好地学习 Android 开发。

**具体功能**

1. 主页展示优秀技术文章，含 Banner、二级分类等交互
2. 项目页面分类展示开源项目信息及文章
3. 登录、注册、退出功能
4. 技术文章搜索功能
5. 收藏功能

**接口**：https://www.wanandroid.com/blog/show/2

**Java 进阶**：线程、抽象类、异常、IO 操作、接口回调

**开源库**：Banner、BottomNavigationView、ViewPager2、TabLayout、SmartRefreshLayout、AgentWeb

**知识点**

- Cookie（OkHttp、Retrofit）
- Socket（局域网通讯，硬件比赛可能用到）
- Layout 中 drawable 资源文件（提升 UI）

> 暑期考核知识点：《第一行代码》+ 上述开源库

### 建议汇总

- 考研/转专业不要落下课内学业
- 闪退多半是空指针异常
- 书上内容可能过时，先百度，无果再问学长
- 原生控件不够用时，学习自定义 View 与组合控件
- 暑期规划好学习计划，劳逸结合，最重要是暑期考核

### 学有余力

MVC、MVP、MVVM 架构，Retrofit 网络请求，MVVM 写项目，Jetpack 常用组件（《第一行代码》第三版），Room 数据库。
