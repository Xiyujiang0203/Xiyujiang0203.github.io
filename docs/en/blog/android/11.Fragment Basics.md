# Fragment Basics

## 1. Definition of Fragments

Fragment: a UI segment embedded within an Activity

---

## 2. Fragment Lifecycle

*Called when creating a fragment*

- onAttach

- onCreate

- onCreateView

- onActivityCreated

- onStart

- onResume

*Called when destroying a fragment*

- onPause

- onStop

- onDestroyView

- onDestroy

- onDetach

---

## 3. Static Fragment Loading

1. Write the fragment layout: create a layout/***.xml file

2. Write the fragment code:

```
public class FirstFragment extends Fragment {//Implement the fragment class
    @Nullable
    @Override
    public View onCreateView(@NonNull LayoutInflater inflater, @Nullable ViewGroup container, @Nullable Bundle savedInstanceState) {
        return inflater.inflate(R.layout.first_fragment,container,false);//Load the child fragment layout into the container; the container is the layout referenced by the name attribute
    }
}
```

3. Reference the fragment in the Activity layout

```

    <fragment
        android:id="@+id/fragment_first"
        android:name="com.example.broadcast.FirstFragment"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        />
```

---

## 4. Dynamic Fragment Loading

1. Create a second fragment layout

2. Write the second fragment code

3. Replace the fragment reference in the Activity with a layout container

```
<FrameLayout
          android:id="@+id/framelayout"
          android:layout_width="match_parent"
          android:layout_height="match_parent"
          />
```

4. Write the fragment-adding logic in the Activity

```
    public void changeFragment(Fragment fragment){
        FragmentManager fragmentManager = getSupportFragmentManager();//Get the FragmentManager
        FragmentTransaction fragmentTransaction = fragmentManager.beginTransaction();//Get the FragmentTransaction
        fragmentTransaction.add(R.id.framelayout,fragment);//Add the fragment to the layout
        fragmentTransaction.commit();//Commit the transaction
    }
```

5. Add the fragment

```
changeFragment(new FirstFragment());//FirstFragment is the fragment class you wrote
```

## 5. Interaction Between Fragments and Activities

1. Get the Activity object from a fragment

```
getActivity()//Get the Activity object so you can use code from the Activity
```

2. Get a fragment object from an Activity

```
fragmentManager.findFragmentById(R.id.fragment_first);//Get the fragment object so you can call its methods
```

3. Communication between fragments

```
getActivity().getSupportFragmentManager().findFragmentById()//Get the Activity, then use it to obtain another fragment
```

## 6. Implementing a Back Stack with Fragments

---

```
fragmentTransaction.addToBackStack(null);//Call this before committing; null is the stack name description
```
