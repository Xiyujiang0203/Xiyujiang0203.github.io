## Enter the code and replace [AK from official site] and [SK from official site] with the API Key and Secret Key from the official console

```python
# encoding:utf-8
import requests 

# client_id is AK from console; client_secret is SK
host = 'https://aip.baidubce.com/oauth/2.0/token?grant_type=client_credentials&client_id=[YOUR_AK]&client_secret=[YOUR_SK]'
response = requests.get(host)
if response:
    print(response.json())
```

> Note: Install requests via `pip install requests` in PyCharm. pip is a basic package manager — search online if you are unfamiliar with it.

## Run the program — the returned access_token will appear in the Run window
