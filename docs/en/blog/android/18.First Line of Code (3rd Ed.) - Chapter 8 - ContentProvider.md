# First Line of Code (3rd Ed.) - Chapter 8 - ContentProvider

## Chapter 8 Content Providers

### 8.1 ContentProvider Overview

ContentProvider: enables data sharing between apps with a complete mechanism for one app to access another's data securely.

### 8.2 Runtime Permissions

#### 8.2.1 How Android Permissions Work

- After requesting permissions, they appear in Settings and the app info screen.

- From Android 6.0, users can grant permissions when a feature is used instead of blocking the app entirely.

- Permissions are normal or dangerous. Normal permissions are granted automatically when declared in the manifest. Dangerous permissions require runtime approval.

- Granting one dangerous permission in a group may auto-grant others in the same group — do not rely on this in code; groups can change.

- All dangerous permissions as of Android 10

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
        makeCall.setOnClickListener {
            if(ContextCompat.checkSelfPermission(this,android.Manifest.permission.CALL_PHONE) != PackageManager.PERMISSION_GRANTED)
            {
                ActivityCompat.requestPermissions(this, arrayOf(android.Manifest.permission.CALL_PHONE),1)
            }else call()
        }
    }

    override fun onRequestPermissionsResult(
        requestCode: Int,
        permissions: Array<out String>,
        grantResults: IntArray
    ) {
        super.onRequestPermissionsResult(requestCode, permissions, grantResults)
        when (requestCode) {
            1->{
                if (grantResults.isNotEmpty() && grantResults[0] == PackageManager.PERMISSION_GRANTED) {
                    call()
                }else{
                    Toast.makeText(this,"You denied the permission",Toast.LENGTH_SHORT).show()
                }
            }
        }
    }

    private fun call() {
        try {
            val intent = Intent(Intent.ACTION_CALL)
            intent.data = Uri.parse("tel:10086")
            startActivity(intent)
        } catch (e: SecurityException) {
            e.printStackTrace()
        }

    }
}

### 8.3 Accessing Another App's Data

- Two ways to use ContentProvider: use an existing provider (contacts, SMS, media library) or expose your own for other apps

#### 8.3.1 Basic ContentProvider Usage

- Use `ContentResolver` for CRUD; it exposes CRUD methods

- CRUD here takes a Uri, not a table name. Uri has three parts: scheme + authority + path — which app's which table. Example: `content://com.example.app.provider/table1`

- Parse the string with `Uri.parse("content://com.example.app.provider/table1")`

- Query API is similar to SQLite with slightly different parameters

- `query(uri, columns, selection, selectionArgs, sortOrder)` returns a Cursor

Read

```
while(cursor.moveToNext()){
	val column1 = cursor.getString(cursor.getColumnIndex("column1"))
	val column2 = cursor.getString(cursor.getColumnIndex("column2"))
}
```

Insert

```
val values = contentValuesOf("column1" to "text","column2" to 1)
contentResolver.insert(uri,values)
```

Clear column1

```
val values = contentValuesOf("column1" to "")
contentResolver.update(uri,values,"column1 = ? and column2 = ?",arrayOf("text","1"))
```

Delete

```
contentResolver.delete(uri,"column2 = ?",arrayOf("1"))
```

#### 8.3.2 Reading System Contacts

1. Prepare a ListView

2. Code

class MainActivity : AppCompatActivity() {

    private val contactList = ArrayList<String>()
    private lateinit var adapter: ArrayAdapter<String>

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
        adapter = ArrayAdapter(this, androidx.appcompat.R.layout.support_simple_spinner_dropdown_item,contactList)
        contactView.adapter = adapter
        if(ContextCompat.checkSelfPermission(this,android.Manifest.permission.READ_CONTACTS) != PackageManager.PERMISSION_GRANTED)
        {
            ActivityCompat.requestPermissions(this, arrayOf(android.Manifest.permission.READ_CONTACTS),1)
        }else readContacts()
    }

    @SuppressLint("Range")
    private fun readContacts() {
        contentResolver.query(ContactsContract.CommonDataKinds.Phone.CONTENT_URI,
            null,null,null,null)?.apply {
                while(moveToNext()){
                    val displayName = getString(getColumnIndex(
                        ContactsContract.CommonDataKinds.Phone.DISPLAY_NAME))
                    val number = getString(getColumnIndex(
                        ContactsContract.CommonDataKinds.Phone.NUMBER))
                    contactList.add("$displayName\n$number")
                }
            adapter.notifyDataSetChanged()
            close()
        }

    }

    override fun onRequestPermissionsResult(
        requestCode: Int,
        permissions: Array<out String>,
        grantResults: IntArray
    ) {
        super.onRequestPermissionsResult(requestCode, permissions, grantResults)
        when (requestCode) {
            1->{
                if (grantResults.isNotEmpty() && grantResults[0] == PackageManager.PERMISSION_GRANTED) {
                    readContacts()
                }else{
                    Toast.makeText(this,"You denied the permission",Toast.LENGTH_SHORT).show()
                }
            }
        }
    }
}

- onCreate(): called when the ContentProvider is initialized; usually creates/upgrades the database. Return true on success, false on failure.

- query(): query data. uri selects the table; projection selects columns; selection and selectionArgs filter rows; sortOrder sorts results. Returns a Cursor.

- insert(): insert a row. uri selects the table; values holds the data. Returns a Uri for the new row.

- update(): update existing data. uri selects the table; values holds new data; selection and selectionArgs filter rows. Returns affected row count.

- delete(): delete data. uri selects the table; selection and selectionArgs filter rows. Returns deleted row count.

- getType(): return the MIME type for the given content Uri.

- Parse the incoming Uri to determine which table and data the caller needs

1. Two Uri formats: a whole table, or one row by id

```
content://com.example.app.provider/table1
content://com.example.app.provider/table1/1
```

1. Two wildcards

- `*` matches any length of any characters

- `#` matches any length of digits

- Uri patterns with wildcards: match any table; match any row in a table

```
content://com.example.app.provider/*
content://com.example.app.provider/table1/#
```

#### 8.5.2 Cross-App Data Sharing

1. Open DatabaseTest, create a ContentProvider: New → Other → Content Provider

```
class DatabaseProvider : ContentProvider() {

    private val bookDir = 0
    private val bookItem = 1
    private val categoryDir = 2
    private val categoryItem = 3
    private val authority =  "com.example.databasetest.provider"
    private var dbHelper: MyDatabaseHelper? = null

    /**
     * by lazy: code block does not run until uriMatcher is first accessed
     * Last line of the block is the return value
     */
    private val uriMatcher by lazy{
        val matcher = UriMatcher(UriMatcher.NO_MATCH)
        matcher.addURI(authority, "book", bookDir)
        matcher.addURI(authority, "book/#", bookItem)
        matcher.addURI(authority, "category", categoryDir)
        matcher.addURI(authority,"category/#",categoryItem)
        matcher
    }

    /**
     * Getter syntax sugar
     * ?. safe call
     * let
     * ?: Elvis
     * Single-expression function
     */
    override fun onCreate() = context?.let {
        dbHelper = MyDatabaseHelper(it, "BookStore.db", 3)
        true
    }?: false

    override fun query(uri: Uri, projection: Array<String>?, selection: String?
                       , selectionArgs: Array<String>?, sortOrder: String?) = dbHelper?.let {
                           val db = it.readableDatabase
        val cursor = when (uriMatcher.match(uri)) {
            bookDir -> db.query("Book", projection, selection, selectionArgs, null, null, sortOrder)
            bookItem -> {
                val bookId = uri.pathSegments[1]
                db.query("Book", projection, "id = ?", arrayOf(bookId), null, null, sortOrder)
            }
            categoryDir -> db.query("Category", projection, selection, selectionArgs, null, null, sortOrder)
            categoryItem -> {
                val categoryId = uri.pathSegments[1]
                db.query("Category", projection, "id = ?", arrayOf(categoryId), null, null, sortOrder)
            }
            else -> null
        }
        cursor
    }

    override fun insert(uri: Uri, values: ContentValues?) = dbHelper?.let {
        val db = it.writableDatabase
        val uriReturn = when (uriMatcher.match(uri)) {
            bookDir,bookItem ->{
                val newBookId = db.insert("Book",null,values)
                Uri.parse("content://$authority/book/$newBookId")
            }
            categoryDir,categoryItem ->{
                val newCategoryId = db.insert("Category",null,values)
                Uri.parse("content://$authority/book/$newCategoryId")
                }
            else -> null
        }
        uriReturn
    }

    override fun update(uri: Uri, values: ContentValues?, selection: String?, selectionArgs: Array<String>?) = dbHelper?.let {
        val db = it.writableDatabase
        val updateRows = when (uriMatcher.match(uri)) {
            bookDir -> db.update("Book", values, selection, selectionArgs)
            bookItem -> {
                val bookId = uri.pathSegments[1]
                db.update("Book", values, "id = ?", arrayOf(bookId))
            }
            categoryDir -> db.update("Category", values, selection, selectionArgs)
            categoryItem -> {
                val categoryId = uri.pathSegments[1]
                db.update("Category",values,"id = ?", arrayOf(categoryId))
            }
            else -> 0
        }
        updateRows
    } ?: 0

    override fun delete(uri: Uri, selection: String?, selectionArgs: Array<String>?) = dbHelper?.let {
        val db = it.writableDatabase
        val deletedRows = when (uriMatcher.match(uri)) {
            bookDir -> db.delete("Book", selection, selectionArgs)
            bookItem -> {
                val bookId = uri.pathSegments[1]
                db.delete("Book", "id = ?", arrayOf(bookId))
            }
            categoryDir -> db.delete("Category", selection, selectionArgs)
            categoryItem ->{
                val categoryId = uri.pathSegments[1]
                db.delete("Category","id = ?", arrayOf(categoryId))
            }
            else -> null
            }
            deletedRows
        } ?: 0

    override fun getType(p0: Uri) = when(uriMatcher.match(p0)){
        bookDir -> "vnd.android.cursor.dir/vnd.com.example.databasetest.provider.book"
        bookItem -> "vnd.android.cursor.item/vnd.com.example.databasetest.provider.book"
        categoryDir -> "vnd.android.cursor.dir/vnd.com.example.databasetest.provider.category"
        bookItem -> "vnd.android.cursor.item/vnd.com.example.databasetest.provider.category"
        else -> null
    }

}
```

class MainActivity : AppCompatActivity() {

    var bookId:String? = null

    private val TAG = "MainActivity"

    @SuppressLint("Range")
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        addData.setOnClickListener {
            val uri = Uri.parse("content://com.example.databasetest.provider/book")
            val values = contentValuesOf("name" to "A Clash of Kings","author" to "George Martin","pages" to 1040,"price" to 22.85)
            val newUri = contentResolver.insert(uri, values)
            bookId = newUri?.pathSegments?.get(1)
        }

        queryData.setOnClickListener {
            val uri = Uri.parse("content://com.example.databasetest.provider/book")
            contentResolver.query(uri,null,null,null,null)?.apply {
                while (moveToNext()) {
                    val name = getString(getColumnIndex("name"))
                    val author = getString(getColumnIndex("author"))
                    val pages = getString(getColumnIndex("pages"))
                    val price = getDouble(getColumnIndex("price"))
                    Log.d(TAG, "book name is $name")
                    Log.d(TAG, "book author is $author")
                    Log.d(TAG, "book pages is $pages")
                    Log.d(TAG, "book price is $price")
                }
                close()
            }
        }

        updateData.setOnClickListener {
            bookId?.let {
                val uri = Uri.parse("content://com.example.databasetest.provider/book/$it")
                val values = contentValuesOf("name" to "A Storm of Swords","pages" to 1216,"price" to 24.05)
                contentResolver.update(uri,values,null,null)
            }
        }

        deleteData.setOnClickListener {
            bookId?.let {
                val uri = Uri.parse("content://com.example.databasetest.provider/book/$it")
                contentResolver.delete(uri,null,null)
            }
        }
    }
}

### 8.5 Kotlin Classroom: Generics and Delegation

#### 8.5.1 Basics of Generics

Definition: generics let you program without specifying concrete types

- Defining your own generic:

  - Generic class:

  ```
  class MyClass<T>{

  	fun method(param: T): T{
  		return param
  	}
  }
  ```

  - Usage: specify the type when instantiating

  ```
  val myClass = MyClass<Int>()
  val result = myClass.method(123)
  ```

  - Generic method:

  ```
  class MyCass{
  	fun <T> method(param: T):T{
  		return param
  	}
  }
  ```

  - Usage

  ```
  val myClass = MyClass()
  val result = myClass.method(Int)(123)
  ```

  - Type inference optimization

  ```
  val myClass = MyClass()
  val result = myClass.method(123)
  ```

  - Note: bound generics with an upper bound

  ```
  class MyCass{
  	fun <T : Number> method(param: T):T{
  		return param
  	}
  }
  //T can only be float, double, int; String would error
  ```

  - Note: default upper bound is Any? (nullable); use Any for non-null

- Optimize the higher-order function from 6.5.1

  - Limitation: build only works on StringBuilder

  ```
  fun StringBuilder.build(block: StringBuilder.() -> Unit):StringBuilder{
      block()
      return this
  }
  ```

  - Generics work on any class

  New build.kt:

  ```
  fun <T> T.build(block: T.() -> Unit):T{
      block()
      return this
  }
  ```

  ```
  contentResolver.query(uri,null,null,null,null)?.build {
      while (moveToNext()) {
     		...
      }
      close()
  }
  ```

- Two kinds of lazy initialization

  - `lateinit`: manual init when first used

  - `by lazy`: automatic init on first access

#### 8.5.2 Class Delegation

Definition: an object delegates work to a helper instead of handling logic itself

- Types

  - Class delegation: one class delegates to another

  Implement Set by delegation:

  ```
  class MySet<T>(val helperSet: HashSet<T>) : Set<T> {
  	override val size: Int
  		get() = helperSet.size
		
  	override fun contains(element: T) = helperSet.contains(element)
	
  	override fun containsAll(element: Collection<T>) = helperSet.containsAll(elements)
	
  	override fun isEmpty() = helperSet.isEmpty()
	
  	override fun iterator() = helperSet.iterator()
  }
  ```

  Optimize with `by`:

  ```
  class MySet<T>(val helperSet: HashSet<T>) : Set<T> by helperSet{
  	//New method
  	fun helloWorld() = println("Hello World")
	
  	//Override
  	override fun isEmpty() = false
  }
  ```

  A new data structure with all HashSet behavior plus custom methods and overrides

  - Property delegation: property implementation delegated to another class

  Syntax:

  ```
  class MyClass{
  	//Property p delegated to Delegate; get/set use Delegate methods
  	//Implement Delegate separately
  	var p by Delegate()
  }
  ```

  ```
  class Delegate{
	
  	var propValue:Any? = null
  	//First param: class using delegation
  	//Second param: property metadata (required but unused here)
  	//<*> means unknown generic type
  	operator fun getValue(myClass: MyClass,prop: KProperty<*>):Any?{
  		return propValue
  	}
	
  	operator fun setValue(myClass: MyClass,prop: KProperty<*>, value: Any?){
  		propValue = value
  	}
  }
  ```

#### 8.5.3 Implementing Your Own lazy Function

- by lazy syntax

```
val p by lazy{ ... }
```

lazy creates and returns a delegate; reading p calls getValue, which runs the lambda; the last line is returned

- Top-level function: function not inside any class

- Custom lazy loader `later`

```
class Later<T>(val block:()->T){
    var value:Any? = null

    operator fun getValue(any:Any?,prop: KProperty<*>):T{
        if(value == null){
           value = block
        }
        return value as T
    }
}

fun <T> later(block: () -> T) = Later(block)
```

Use a button click to verify: first click logs, later clicks do not; no click means no init — init on first use

```
val p by later{
	Log.d("TAG","run codes inside later bloce")
	"test later"
}
```
