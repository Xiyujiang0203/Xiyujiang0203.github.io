# Android Learning Roadmap (2022)

Android development aims to build apps that run on the Android system using Java or Kotlin on the Android Studio platform.

## I. First Semester, Freshman Year

### 1. Complete Java SE

Covers Java basic data types, branching, loops, arrays, classes and objects, wrapper classes and common methods, encapsulation, inheritance, polymorphism, abstract classes, interfaces, exceptions, I/O, threads, and more.

### 2. Study *Android First Line of Code*, 2nd Edition

- Set up the Android development environment and understand project structure
- Master Activity: lifecycle, launch modes, and Intent usage
- Proficiently use common widgets and layouts; learn ListView and RecyclerView
- Master Fragment, broadcasts, and multimedia
- Learn SQLite, SharedPreferences, and the LitePal database

### 3. Milestone Assessment

Build a simple app with clear functionality. (Used for the Lanqiao Cup assessment)

## II. Second Semester, Freshman Year

### 1. Reinforce Java SE

Become proficient in Java.

### 2. Complete *Android First Line of Code*, 2nd Edition

- Use SQLite and Material Design
- Master Android networking: OkHttp requests, GSON JSON parsing, and API data handling
- Master ContentProvider and runtime permissions
- Master Android threading and Services

### 3. Milestone Assessment

Build a practical, feature-complete app with reasonable project structure that demonstrates key skills. (Used for the summer assessment)

## III. First Semester, Sophomore Year

### 1. Study *Android First Line of Code*, 3rd Edition

Learn Android development with Kotlin.

### 2. Source Code and Custom Views

Read Framework-layer source code and advanced widget source (e.g., ListView, RecyclerView); try building custom Views.

### 3. Advanced Topics

Learn Android animations and design patterns.

### 4. Official Documentation

Read the Android official documentation.

### 5. Milestone Goal

Gain basic independent development ability.

## IV. Later Stages

After that, plan your own learning path based on your skills and project experience.

## Recommended Books

- *Android First Line of Code*, 2nd Edition
- *Android First Line of Code*, 3rd Edition
- *Java from Beginner to Expert*
- *Core Java*
- *Android Development Art Exploration*
- *Android Source Code Design Patterns Analysis and Practice*

## Supplementary Notes

### Freshman Year, First Semester · Environment Setup

- Install IntelliJ IDEA CE (from the official site) — Java build environment
- Install JDK 1.8 (can be downloaded from within the IDE)
- For detailed setup, search CSDN or ask senior students
- Android Studio: https://developer.android.google.cn
- Configure the SDK (usually during install; if download fails due to network issues, ask seniors for a pre-downloaded SDK)

### Freshman Year, First Semester · Learning Tips

- Book: *Java from Beginner to Expert*
- Video: Bilibili Black Horse Programmer Java course (pick one that suits you)
- Follow along by typing, but you don't need to type everything — understanding matters most
- About 1–2 weeks for Java basics; complete overall tasks within two weeks after joining the tech association
- The track itself doesn't require notes, but association note tasks must be completed
- Aim to understand; forgetting quickly is fine — look things up on CSDN when needed and you'll get familiar through practice
- Skipping ahead while reading risks confusion — check whether you skipped prerequisite content
- Use Typora to record your progress for review and code reuse
- Don't just read — running the book's code is important
- Don't fear forgetting; learning is iterative (e.g., adapters may need several re-reads)

### Winter Break

Continue studying *Android First Line of Code*.

**Suggestion**: Before sophomore year starts, run all code in the book and save records in Typora. Chapters 11 and 15 can be skipped; Chapter 14 doesn't need full implementation (many bugs), but study its architecture patterns.

### Freshman Year, Second Semester · Practice Project: WanAndroid Platform

**Description**: After two semesters of Android learning, build an app to help beginners learn Android development.

**Features**

1. Home page showing quality tech articles with Banner, secondary categories, and other interactions
2. Project page categorizing open-source projects and articles
3. Login, register, and logout
4. Tech article search
5. Favorites

**API**: https://www.wanandroid.com/blog/show/2

**Java Advanced Topics**: threads, abstract classes, exceptions, I/O, interface callbacks

**Open-Source Libraries**: Banner, BottomNavigationView, ViewPager2, TabLayout, SmartRefreshLayout, AgentWeb

**Key Topics**

- Cookie (OkHttp, Retrofit)
- Socket (LAN communication; may be useful for hardware competitions)
- Drawable resources in layouts (better UI)

> Summer assessment scope: *Android First Line of Code* + the libraries above

### Summary of Suggestions

- If pursuing grad school or changing majors, don't neglect coursework
- Crashes are often NullPointerException
- Book content may be outdated — search Baidu first, then ask seniors
- When built-in widgets aren't enough, learn custom Views and composite widgets
- Plan summer study well, balance work and rest — the summer assessment matters most
